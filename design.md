# 台灣公園探索 - 介面設計計畫

## 設計理念

以自然、戶外探索為主題，採用清新綠色系搭配溫暖大地色，營造親近自然、適合親子出遊的氛圍。整體遵循 Apple HIG 設計規範，以直覺操作為核心。

## 頁面清單

### 1. 首頁
- **路徑**: `(tabs)/index.tsx`
- **內容**: 
  - 頂部 App 標題列 + 縣市選擇器
  - 分類快速入口（5 個分類卡片，橫向排列）
  - 熱門公園推薦區（好玩指數最高的公園卡片）
  - 最新評論動態
- **功能**: 點擊分類跳轉搜尋頁並套用篩選；點擊公園卡片進入詳情頁

### 2. 搜尋頁
- **路徑**: `(tabs)/search.tsx`
- **內容**:
  - 搜尋輸入框（公園名稱關鍵字）
  - 縣市篩選下拉選單
  - 分類篩選標籤列（可複選）
  - 好玩指數篩選（1-5 星）
  - 搜尋結果列表（FlatList，公園卡片含名稱、地址、分類標籤、好玩指數）
- **功能**: 即時搜尋、多條件篩選、點擊進入詳情頁

### 3. 公園詳情頁
- **路徑**: `park/[id].tsx`
- **內容**:
  - 公園名稱與地址
  - 分類標籤
  - 好玩指數（星星顯示）
  - 特色介紹文字
  - 設施清單
  - 評論列表
  - 新增評論表單（評分 + 評論文字）
  - 導航按鈕（開啟 Google Maps）
  - 收藏按鈕
- **功能**: 查看完整資訊、留下評論、導航、收藏

### 4. 收藏頁
- **路徑**: `(tabs)/favorites.tsx`
- **內容**:
  - 收藏的公園列表（FlatList）
  - 空狀態提示
- **功能**: 查看已收藏公園、點擊進入詳情頁、取消收藏

### 5. 地圖頁
- **路徑**: `(tabs)/map.tsx`
- **內容**:
  - 縣市選擇器
  - 公園列表按距離/地區排列
  - 每個公園項目有「導航」按鈕
- **功能**: 選擇地區瀏覽公園、一鍵導航至 Google Maps

## 關鍵使用者流程

### 流程一：搜尋共融式公園
1. 使用者開啟 App → 首頁
2. 點擊「共融式」分類卡片 → 搜尋頁（已套用共融式篩選）
3. 瀏覽結果列表 → 點擊有興趣的公園
4. 進入詳情頁 → 查看特色與評論
5. 點擊「導航」→ 開啟 Google Maps 導航

### 流程二：留下評論
1. 使用者進入公園詳情頁
2. 滾動至評論區 → 點擊「新增評論」
3. 選擇好玩指數（1-5 星）→ 輸入評論文字
4. 點擊「送出」→ 評論儲存至 AsyncStorage 並即時顯示

### 流程三：收藏公園
1. 使用者在詳情頁點擊愛心圖示
2. 公園加入收藏列表（AsyncStorage）
3. 切換至「收藏」分頁查看已收藏公園
